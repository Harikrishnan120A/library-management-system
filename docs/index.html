Java 17 + Maven + Swing

Library Management System

A desktop-first library app with a Swing GUI, console fallback, overdue tracking, reports, and CSV tools.

View Source Run Locally Project Report Download DOC

GitHub Pages hosts project documentation only. The Swing app runs on your machine, not in the browser.

What It Does

Book CRUD Issue / Return Overdue Detection Fine Calculation CSV Import / Export Dashboard + Reports SMS Reminders Serialization Persistence

Run It

GUI Mode

mvn compile
mvn exec:java -Dexec.mainClass="com.library.Main"

Console Mode

mvn compile
mvn exec:java -Dexec.mainClass="com.library.Main" -Dexec.args="--console"

Stack

Java 17, Maven, Java Swing, java.util.logging, and object serialization for data storage.
